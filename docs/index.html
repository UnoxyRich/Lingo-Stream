Flow

How Lingo Stream works

  1. Watch a YouTube video with captions

    The extension listens for subtitle changes in YouTube’s caption DOM nodes.

  2. Select meaningful words

    Eligible words are chosen based on your replacement percentage and filtering rules.

  3. Translate with provider fallback

    Words are translated using available free endpoints with caching and miss backoff.

  4. Render inline learning cues

    Captions update to formats like enjoy (gusto) in the active subtitle line.

Install Guide

Install the Chrome extension

Option A: Clone with Git

  1. Open your terminal.
  2. Clone the repository:
    git clone https://github.com/your-username/Lingo-Stream.git
  3. Open Chrome and go to chrome://extensions.
  4. Enable Developer mode in the top-right.
  5. Click Load unpacked and select the extension folder.

Option B: Download ZIP

  1. Open the repository on GitHub using the button below.
  2. Click Code then Download ZIP.
  3. Extract the ZIP file on your computer.
  4. Open chrome://extensions and enable Developer mode.
  5. Click Load unpacked and choose the extracted extension folder.
Open Repository

After loading, pin the extension from Chrome’s toolbar, open a YouTube video with captions, and set your language in the popup.

FAQ

Quick answers

Is it on the Chrome Web Store?

Not yet. Current installation is through Developer Mode using Load Unpacked.

Does it cost money?

No. It uses free translation endpoints and does not require API keys by default.

Will this break YouTube captions?

It only modifies caption text content and avoids replacing caption DOM elements.
