Install

Install Lingo Stream in Chrome

Option 1: Download ZIP

  1. Download this package:
  2. Lingo Stream v0.0.1 alpha.zip
  3. Extract the ZIP on your computer.
  4. Open chrome://extensions.
  5. Enable Developer mode.
  6. Click Load unpacked and select the extracted extension folder.

Option 2: Clone from GitHub

  1. Clone the repository:
  2. git clone https://github.com/your-username/Lingo-Stream.git
  3. Open chrome://extensions in Chrome.
  4. Enable Developer mode.
  5. Click Load unpacked and pick the repo's extension folder.

GitHub Pages setup: choose Deploy from a branch, then select main and folder /docs.
