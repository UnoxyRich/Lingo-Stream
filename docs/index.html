Install

Install Lingo Stream in Chrome in minutes

  1. Download the packed ZIP package below.
  2. Extract the ZIP file.
  3. Open chrome://extensions.
  4. Enable Developer mode.
  5. Click Load packed and choose the packaged extension file from the extracted ZIP.
