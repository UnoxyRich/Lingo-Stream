Install

How to install Lingo Stream

Option 1: Download ZIP (quickest)

  1. Download the extension ZIP package:
  2. Lingo Stream v0.0.1 alpha.zip
  3. Extract the ZIP file.
  4. Open chrome://extensions.
  5. Enable Developer mode.
  6. Click Load unpacked and select the extracted extension folder.

Option 2: Clone from GitHub

  1. Clone the repository:
  2. git clone https://github.com/your-username/Lingo-Stream.git
  3. Open chrome://extensions in Chrome.
  4. Enable Developer mode and click Load unpacked.
  5. Select this repository's extension folder.

GitHub Pages deploy (branch mode): In your repository settings, open Pages, then choose Deploy from a branch and set branch to main, folder / (root).
